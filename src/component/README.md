# 컴포넌트

- 리액트(React)에서 컴포넌트(Component)는 UI를 구성하는 독립적인 요소이다. 즉, 화면에 나타나는 부분을 작은 단위로 분할하여 재사용 가능한 모듈로 만들어 놓은 것.

- 리액트 애플리케이션은 여러 개의 컴포넌트로 구성된다. 이러한 컴포넌트들은 다른 컴포넌트들을 포함하거나 서로의 기능을 확장하여 복잡한 UI를 만들 수 있다.
